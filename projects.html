Quiz Master

A web-based quiz platform with admin and user functionalities built using Flask.

Description

The Quiz Master application is a web-based platform designed for the creation, management, and participation in quizzes. It provides functionalities for both administrators and regular users.

Key Features

  • Admin panel for managing subjects, chapters, and questions
  • User registration and authentication
  • Interactive quiz interface
  • Score tracking and performance analytics
  • Graphical representation of results

Technical Approach

  • Flask backend with SQLAlchemy ORM
  • Jinja2 templating for frontend
  • Password hashing for security
  • Matplotlib for data visualization
Flask Python SQLAlchemy Jinja2 Matplotlib HTML/CSS

Fingerprint Electronic Voting System

Biometric voting solution using Arduino and fingerprint authentication.

Description

State-of-the-art solution designed to modernize the electoral process by integrating biometric authentication with traditional voting mechanisms using Arduino Nano and R307 Fingerprint Sensor.

Key Features

  • Biometric voter authentication
  • Real-time vote recording
  • LCD display for user interface
  • GSM module for notifications
  • Secure alarm system

Technical Specifications

  • Arduino Nano microcontroller
  • R307 Fingerprint Sensor
  • 16x2 LCD Display
  • SIM800L GSM Module
  • Custom PCB design

Social Impact

  • Enhances democratic participation
  • Reduces electoral fraud
  • Promotes equal access to voting
  • Aligns with UN Sustainable Development Goals
